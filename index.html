Info zu diesem Produkt:

Das MBD Modul ist ein Türsensor welcher über WiFi mit dem MBD Client kommuniziert. Wenn die Tür an der das MBD Modul angebracht ist geöffnet wird, sendet das Modul einen Befehl an den MBD Client. Dieser schaltet daraufhin das vorher ausgewählte Mikrofon ab, wodurch eine Störung einer Videokonferenz oder sonstigem verhindert werden kann.
